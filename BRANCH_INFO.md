# feature/rfc-upload-portal

This branch contains the RFC Document Upload Portal features:

- `RfcDocUploadController` — Upload RFC docs, generate C# controllers, push to GitHub
- `Views/RfcDocUpload/Index.cshtml` — 5-step upload portal UI
- `Views/RfcDocUpload/Swagger.cshtml` — Live Swagger UI, OpenAPI JSON, Try-It
- `App_Start/RouteConfig.cs` — Startup page set to RfcDocUpload
- Auto-updates Swagger explorer on every push
